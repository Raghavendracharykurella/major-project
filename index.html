Installation

  1. Clone the repository:
    git clone https://github.com/Raghavendracharykurella/major-project.git
  2. Change into the project directory:
    cd major-project
  3. Install dependencies:
    pip install opencv-python mediapipe python-pptx pyautogui

Or use requirements.txt:
pip install -r requirements.txt

Project Code Explanation

# Main code flow
1. Initialize webcam and MediaPipe hand detector
2. Capture frames
3. Detect hand landmarks
4. Recognize gestures
5. Send corresponding command to presentation software
6. Display feedback

The application leverages MediaPipe for hand-tracking, reads webcam frames using OpenCV, counts fingers and identifies gestures, then triggers PyAutoGUI and python-pptx actions to control slides.
